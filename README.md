# Lista de contatos

Projeto de uma Lista de contatos desenvolvido utilizando JSP, Servlets.

## Funcionalidades

**Cadastro de Contatos**
**Visualização e Atualização de Contatos**
**Exclusão de Contatos**

## Tecnologias Ultilizadas

**Java**
**JSP**
**Servlets**
**JDBC**
**MySQL**
