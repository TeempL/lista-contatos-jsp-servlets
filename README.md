# Lista de contatos

Projeto de uma Lista de contatos desenvolvido utilizando JSP, Servlets.

## Funcionalidades

1.**Cadastro de Contatos**

2.**Visualização e Atualização de Contatos**

3.**Exclusão de Contatos**

## Tecnologias Ultilizadas

1.**Java: 17**

2.**JSP**

3.**Servlets**

4.**JDBC**

5.**MySQL**
